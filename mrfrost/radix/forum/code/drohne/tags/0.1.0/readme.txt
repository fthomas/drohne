
  Installation
  ------------

    Benutze die Datei "Drohne.exe" um das Programm zu starten.
    Sollte das Programm mit einer Fehlermeldung abbrechen, so
    muss das Microsoft .NET Framweork nachinstalliert werden.

    Das .NET Framework kann über die "Update" Funktion im
    Startmenü, oder von folgendem Link bezogen und installiert
    werden:
    
    http://www.microsoft.com/downloads/details.aspx?FamilyID=262d25e3-f589-4842-8157-034d1e7cf3a3&DisplayLang=en
